# 🌀 Animated Shader: Neon Spiral Loops

A hypnotic flow of neon spirals in motion—where light, math, and digital art blend into an endless glowing rhythm.

Move your cursor to warp the neon spiral and click fullscreen for full immersion.

---

## ✨ Features

- 🌈 Dynamic neon spiral animation  
- 🖱️ Interactive cursor-based distortion  
- ⤢ Fullscreen immersive mode  
- 🎨 Smooth shader-like visual effects  
- ⚡ Lightweight and fast rendering  

---

## 🎮 Controls

- **Move Cursor** → Warp and distort the spiral  
- **Click (⤢)** → Toggle fullscreen mode  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript  
- Canvas / WebGL (Shader-based animation)

---

## 📂 Project Structure

```
project-folder/
│

```

---

## 🚀 How to Run

1. Clone or download this repository  
2. Keep all files in the same directory  
3. Open `index.html` in your browser  
4. Enjoy the neon spiral experience ✨  

---

## 🎯 Project Idea

This project explores the beauty of mathematical motion and digital art through animated neon spirals, creating a relaxing and visually engaging experience.
